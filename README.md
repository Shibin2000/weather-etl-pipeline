# Weather ETL Pipeline

A data engineering project that extracts live weather data, 
transforms it, and loads into a DuckDB warehouse.

## What it does
- Extracts real-time weather data from OpenWeatherMap API
- Transforms and categorizes temperature data using Python
- Loads into DuckDB data warehouse
- Analyzes with SQL queries
- Visualizes with interactive Plotly dashboard

## Tools Used
- Python
- Pandas
- DuckDB
- Plotly
- OpenWeatherMap API
- Google Colab

## Key Findings
- Tracked 5 US cities in real time
- Austin was hottest at 63.6°F
- Chicago was coldest at 24.6°F
- Average temperature: 48.68°F
